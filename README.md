# 🚀 ProTask Manager 2.0

A modern and powerful **Task Management Web Application** built using HTML, CSS, and JavaScript.  
ProTask Manager helps users organize tasks efficiently with features like priority levels, reminders, progress tracking, and dark mode.

---

## ✨ Features

- ✅ Add, Edit, and Delete Tasks
- 📂 Task Categories (Personal, Work, Study)
- 🚩 Priority Levels (Low, Medium, High)
- 📅 Due Date and Time Reminders
- 🔔 Browser Notifications for task alerts
- 🔍 Search Tasks instantly
- 🎯 Filter Tasks (All / Pending / Completed)
- 📊 Progress Bar for completed tasks
- 🌙 Dark / Light Mode toggle
- 🔄 Drag and Drop task reordering
- 💾 Local Storage support (tasks saved in browser)

---

## 🛠 Tech Stack

- HTML5
- CSS3
- JavaScript (ES6)
- Tailwind CSS
- LocalStorage API
- Notification API
